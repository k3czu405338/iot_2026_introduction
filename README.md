# Arduino WiFi Example

This project connects Arduino Uno WiFi Rev2 to WiFi and prints:
- IP address
- MAC address
- Signal strength

## Requirements
- Arduino Uno WiFi Rev2
- WiFiNINA library

## Setup
1. Install WiFiNINA in Arduino IDE
2. Edit SSID and password in code
3. Upload sketch
4. Open Serial Monitor (9600 baud)

## Output
Displays connection status and network info.
